Chicago Public Library - Search Bookmarklet

Introduction

This browser bookmarklet allows you to easily search the Chicago Public Library (CPL) catalogue for any text displayed on a web page.

Installation

To install the bookmarket, click and drag the button named 'Search CPL' below to your browser bookmarks bar.

Usage

Once the bookmarlet has been installed, you can search the CPL catalogue with the following steps:

1) Highlight the text to search on any web page.

2) Click the 'Search CPL' bookmarklet on your browser bookmarks bar.

3) The CPL catalogue search results will be displayed in a new browser tab.
