Chicago Public Library - Search Bookmarklet

Introduction

This browser bookmarklet allows you to easily search the Chicago Public Library (CPL) catalogue for any text on a web page.

Installation

To install the bookmarket, click and drag the button named 'Search CPL' below to your browser bookmarks.

Usage

Once the bookmarlet has been installed, you can search the CPL catalogue with the following steps:

1) Highlight the text on any web page.

2) Click the 'Search CPL' bookmarklet.
